# iWorld Store POS

Point of Sale software for managing phone and iPhone inventory.

## Features

- ✅ Add Inventory (IMEI scanning)
- ✅ Sell Phones
- ✅ Return/Refund Processing
- ✅ Inventory Management
- ✅ Profit Reports
- ✅ Export to CSV/Excel

## Tech Stack

- React + TypeScript
- Express.js
- SQLite (better-sqlite3)
- Material-UI

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed on Vercel via GitHub.
